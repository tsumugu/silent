# v1.0.38 (2026-01-02)

## 概要
今回のリリースでは、アプリケーションの基盤となるリアーキテクチャの推進に加え、アーティスト詳細画面の追加や、操作性向上のための Escape キーの挙動修正を行いました。

## 新機能
- **アーティスト詳細画面の追加**: アーティストの楽曲やアルバムを一覧表示できる詳細ビューを実装しました (#4)
- **リアーキテクチャの推進**: 
  - **Phase 1**: コアタイプの定義とデータマッパーの刷新 (#8)
  - **Phase 2**: ナビゲーション管理の Zustand 移行と、データキャッシュの安定化 (#9)

## 改善・修正
- **操作性の向上**:
  - `Escape` キーの挙動を修正しました。意図しないバックナビゲーションを抑制し、プレイヤーのクローズや検索のクリア機能のみに限定しました。
  - 戻るボタンのツールチップから混乱を招く表記を削除しました。
- **再生機能の安定化**:
  - アルバムやプレイリストの連続再生機能における不具合を修正しました (#1)
  - 再生ステータスの同期処理をより堅牢に改善しました (#2)
- **UI/デザインのブラッシュアップ**:
  - アートワークの背景調整や、シャドウの表示崩れの修正を行いました。
  - 楽曲情報の表示幅制限を撤廃し、長いタイトルも適切に表示されるようにしました (#3)
  - 詳細画面の戻るボタンをスティッキー（固定）表示に変更しました (#5)
- **システム修正**:
  - 「このアプリについて」画面のバージョン表示を動的に取得するように修正しました (#6)
  - macOS において Keychain のアクセス許可プロンプトが表示される問題を抑制するため、Cookie の暗号化を一部無効化しました。
